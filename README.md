# QRCodeZ Generator

Generate QR codes quickly and easily with QRCodeZ Generator. A simple and intuitive web application to create custom QR codes for any text, URL, or data.

## Features

- Generate QR codes from any text or URL
- Easy to use interface
- instant download
- Customizable QR codes

## Installation

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

Or

```bash
yarn install
```

## Usage

Start the development server:

```bash
npm start
```

Or

```bash
yarn start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm test` - Launches the test runner
- `npm run build` - Builds the app for production
- `npm run eject` - Ejects from Create React App (irreversible)

Or

- `yarn start` - Runs the app in development mode
- `yarn test` - Launches the test runner
- `yarn build` - Builds the app for production
- `yarn eject` - Ejects from Create React App (irreversible)
